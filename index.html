Secuencia de Instalación

El sistema multimedia de BYD está protegido contra instalaciones externas por defecto. Para romper este bloqueo, es obligatorio seguir este orden:

1
MÉTODO WI-FI: Se utiliza inicialmente para instalar la "llave" (unLocker) de forma inalámbrica.
2
LIBERTAD TOTAL: Una vez activado el unLocker, el sistema permitirá reconocer aplicaciones permanentemente.
3
COEXISTENCIA DE MÉTODOS: Podrá seguir utilizando el Wi-Fi para instalar aplicaciones si lo desea. Este proceso inicial es indispensable para quitar el bloqueo de instalaciones directamente desde el USB.

06

Activación del Unlocked

Luego de haber explicado el proceso de cómo instalar las futuras aplicaciones, necesitamos previamente instalar el archivo Package Installer Unlocked. Este archivo permitirá al usuario poder instalar aplicaciones directamente desde su USB o pantalla descargando las apps desde librerías como APKPURE o APTOIDE sin necesidad de utilizar nuevamente ATV TOOLS.

Atención / Nota Importante

El archivo Package Installer Unlocked es un componente técnico. Al finalizar su instalación, no se verá reflejado con un icono visible en la pantalla; su función es habilitar permisos internos de forma transparente.

usb_flash_drive

Advertencia de Almacenamiento

Cada vez que intente instalar aplicaciones o archivos mediante USB, el dispositivo deberá estar formateado previamente en sistema FAT32 para asegurar su correcta lectura por el vehículo.

menu_book Glosario de Términos

BYD CONNECT
ADB (Android Debug Bridge) Herramienta versátil que permite la comunicación entre una computadora (o smartphone) y un dispositivo Android (el vehículo) para realizar tareas avanzadas de depuración e instalación.
APK / Instalar Formato de archivo utilizado para distribuir e instalar aplicaciones en el sistema operativo Android.
FAT32 Sistema de archivos universalmente compatible con los sistemas multimedia de BYD. Es necesario formatear el USB en este formato para que el auto reconozca los archivos.
IP (Internet Protocol) Dirección numérica única que identifica al vehículo dentro de la red Wi-Fi para que la aplicación del celular sepa a dónde enviar los datos.
IMEI / ID Código único de identificación del hardware del vehículo. Se utiliza como base para generar la clave maestra de acceso técnico.
Package Installer Unlocked Pequeño archivo de sistema que, una vez instalado, elimina permanentemente las restricciones del puerto USB multimedia, permitiendo instalaciones directas en el futuro.
Sideload Proceso de instalación de aplicaciones que no provienen de la tienda oficial preinstalada, realizado mediante herramientas externas como ATV Tools.
