GLOSARIO TÉCNICO BYD CONNECT URUGUAY

menu_book Glosario

BYD CONNECT
ADB (Android Debug Bridge)

Protocolo de comunicación que permite a dispositivos externos enviar comandos de nivel de sistema al vehículo para instalaciones avanzadas.

APK / Archivo de Instalación

Paquete de software que contiene todos los datos de una aplicación Android, listo para ser ejecutado e instalado manualmente.

FAT32 (File Allocation Table 32)

Formato de sistema de archivos requerido para que el multimedia de BYD pueda leer correctamente las unidades USB externas.

Package Installer Unlocked

Componente crítico que, al instalarse, elimina las restricciones de seguridad del puerto USB, permitiendo la "Libertad Total" de instalación.

IP (Dirección de Protocolo de Internet)

Identificador numérico único del vehículo en la red Wi-Fi; actúa como la "dirección postal" para enviar aplicaciones desde el celular.

Sideloading

Técnica de instalar aplicaciones de fuentes de terceros fuera de la tienda oficial del fabricante.
