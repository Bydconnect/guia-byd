Secuencia de Instalación Obligatoria

El sistema multimedia de BYD está protegido contra instalaciones externas por defecto. Para romper este bloqueo, debe seguir este orden estricto:

1
MÉTODO WI-FI: Se utiliza inicialmente para instalar la "llave" (unLocker) de forma inalámbrica a través de ADB.
2
LIBERTAD TOTAL: Una vez activado el unLocker, el sistema permitirá reconocer aplicaciones desde USB.

Activación del unLocker

Luego de haber explicado el proceso inalámbrico, necesitamos previamente instalar el archivo PackageInstallerunlocked. Este archivo permitirá instalar aplicaciones directamente desde su USB descargando desde librerías externas.

Atención / Nota Importante

El unLocker es un componente técnico. Al finalizar su instalación, no se verá reflejado con un icono visible; su función es habilitar permisos internos.

usb_flash_drive

Advertencia de Almacenamiento

Cada vez que intente instalar aplicaciones o archivos mediante USB, el dispositivo deberá estar formateado previamente en sistema FAT32.

menu_book Glosario Técnico

BYD CONNECT
ADB (Android Debug Bridge)Protocolo de comunicación técnica para enviar comandos desde un smartphone externo al sistema del vehículo inalámbricamente.
APKFormato de archivo ejecutable utilizado para distribuir e instalar aplicaciones en sistemas Android.
FAT32Sistema de archivos obligatorio para que el sistema multimedia BYD reconozca y lea memorias USB correctamente.
IP (Internet Protocol)Dirección numérica única del vehículo dentro de la red Wi-Fi para su vinculación técnica.
IMEI / IDIdentificador único de hardware necesario para generar la clave técnica de apertura en la Fase 1.
unLockerArchivo que elimina permanentemente las restricciones nativas de BYD para puertos USB.
